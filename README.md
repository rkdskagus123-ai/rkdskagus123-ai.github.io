# Namhyun Kang Portfolio

Static portfolio site for `namhyunkang.github.io`.

## Contents

- `index.html`: portfolio landing page
- `styles.css`: site styles
- `resume.html`: public resume page
- `case-studies.html`: public case studies page
- `redmine-case-study.md`: short sanitized Redmine automation sample
- `assets/*.png`: public-safe evidence snapshots

## Notes

This repository intentionally contains only public-safe portfolio materials. Private company URLs, internal IPs, real customer data, real child photos, and real exam/student data are excluded.

The site uses text-first evidence blocks and public-safe evidence snapshots. Real product screenshots can be added later when sanitized demo data is ready.
