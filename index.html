01

Python automation · work tool

Redmine/GitLab 주간보고 자동화

문제

매주 Redmine wiki 개인 섹션을 수동 작성하고, GitLab 커밋 이력을 별도로 확인하느라 기록/작성까지 1시간 이상 걸렸습니다.

구현

Markdown outline을 wiki 본문으로 변환하고, 상태/날짜/진행률 누락을 게시 전에 검증해 10분 미만 리뷰 중심 흐름으로 줄였습니다.

증거

python redmine_weekly_report.py --validate
확인 질문 없음: 상태, 진행률, 날짜가 모두 채워져 있습니다.

기술

  • Python
  • Redmine API
  • GitLab API
  • Markdown
  • PowerShell
